Employee API Test Suite
Automated Postman tests for the Dummy REST API.
---How to Run---
1. Import "CRUD Employee Collection.postman_collection .json" into Postman.
2. Import `Employee Environment.postman_environment.json` into Postman.
3. Select Employee Enviornment from the enviornment drop down to run in the top-right corner.
4. Click on CRUD Employee Collection in the sidebar, select Run.
